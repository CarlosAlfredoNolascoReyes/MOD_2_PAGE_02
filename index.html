Demostracion de Box Sizing

Content Box

Este es un Content Box
width: 200px
padding: 20px
border: 5px
Ancho total: 250px

Border Box

Este es un Border Box
width: 200px
padding: 20px
border: 5px
Ancho total: 200px

Explicacion

Content Box: El ancho total es la suma de width + padding + border (200px + 40px + 10px = 250px).

Border Box: El ancho total es exactamente el width especificado (incluye padding y border).
